Correct Order to declare with qualifier

From Transitioning to ARC Release Notes

ClassName * qualifier variableName;

For example:

MyClass * __weak myWeakReference;
MyClass * __unsafe_unretained myUnsafeReference;

Local Variable Initialization

  • Local C type variables like BOOL are not initialized.
  • Instance variables of Objective-C objects are always set to 0 (or nil) when you allocate an object (this includes BOOL).
